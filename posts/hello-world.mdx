I've rebuilt my blog three times this year. Ghost. Notion-as-a-CMS.
A WordPress instance I shut down within a week.

This time it's just markdown files in a `posts/` folder, statically
rendered by Next.js. No CMS, no admin panel, no migrations. The friction
of "writing a post" should be opening a `.mdx` file in my editor.

Goals for this iteration:
- 📝 Posts are plain MDX — git is the CMS
- ⚡ Static generation — every page pre-rendered
- 🎨 Code highlighting at build time (no runtime cost)
- 🌗 Dark mode that respects OS preference

We'll see if I actually post.
